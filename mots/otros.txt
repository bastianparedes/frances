ça / cela / ceci / ce / cet / cette / ces

pronoms démostratifs
ça: this / that. Cuando está antes de "être" o que se convierte en "ce" o "c'"
ceci: this
cela: that

adjetifs démostratifs
ce / cet: este
cette: esta
ces: estos / estas

lorsque: cuando ("cuando me amas, te amo") (es un poco más formal que quand, pero no puede ser usado para preguntar)
